## Less vs Sass/SCSS

Sass:

```sass
body
	font-family: sans-serif
	color: white

h1
	font-size: 2rem
```


SCSS/Less:

```scss
body {
	font-family: sans-serif;
	color: white;
}

h1 {
	font-size: 2rem;
}
```
